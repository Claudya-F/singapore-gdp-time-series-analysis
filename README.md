# Singapore GDP Time Series Analysis

This project analyses Singapore’s GDP growth using time series methods in R, including STL decomposition and ARIMA/ETS models.

## Methods
- STL Decomposition
- ARIMA modelling
- ETS modelling
- Residual diagnostics
- Forecasting

## Tools
- R
- fpp3
- tsibble
- ggplot2
